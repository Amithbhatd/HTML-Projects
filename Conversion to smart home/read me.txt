About project:-
it helps convert normal devices into smart devices by enabling the user to control the device removely using
bluetooth and voice cammands.

Included:- The Report of the project
	   Proteus schematic
	   The Android App used for controlling devices
	   Code for the Arduino


Running simulation:-
To run the simulation first Install proteus software.Open the given schematic. Compile the arduino code 
and copy the location into the arduino block and past in its target location.Check the bluetooth com port 
and enter it into the HC-05 module in the Simulation .Run the simulation then connect the android app to
 the PC and the system should be working.
